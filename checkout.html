FAQ

This prototype focuses on end-to-end storefront flow: persistent bag, product detail, catalogue filtering, and checkout confirmation. It is intentionally static and local-first.

Shipping

Free delivery across the UAE on orders above AED 150. Orders below that threshold use a flat AED 20 shipping fee in this prototype.

Returns

Standard items can be returned within 14 days. Personalised shirts are treated as final sale unless there is a production fault.

Size Guide

Replica tops fit true to size. Trainingwear is slightly more athletic. If you are between sizes, choose the larger size for a relaxed matchday fit.
